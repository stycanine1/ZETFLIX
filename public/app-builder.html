Installation Methods

Android Chrome:

  1. Visit ZETFLIX in Chrome browser
  2. Tap the menu (⋮) in Chrome
  3. Select "Add to Home screen"
  4. Tap "Add" to install the app
  5. Find ZETFLIX icon on your home screen

Android Firefox:

  1. Open ZETFLIX in Firefox
  2. Tap the menu (⋮) in Firefox
  3. Select "Install" option
  4. Confirm installation
  5. Launch from home screen

Automatic Install:

When you visit ZETFLIX, you'll see an "Install App" button. Simply tap it and follow the prompts!
